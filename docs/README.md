# 微信小程序教程 · vibe coding 版

> **人人都能做小程序** —— 全程用 AI（混元3），从 0 到 1 开发、调试、发布一个真正能扫码使用的微信小程序。作者本人**没有手写一行代码**。

---

## 这套教程适合谁？

- 👶 **完全不懂编程**，但想拥有一个自己的小程序
- 🧑‍💼 产品、运营、设计、老师、个体户……有想法但不会写代码
- 🧑‍🎓 想看看「AI 到底怎么帮人写软件」的好奇者

只要你**会打字、有一台电脑、有一个微信**，就能跟着做完。

## 你将做出什么？

一个名为「**微信小程序教程**」的真实小程序，里面有：

- 42 个交互控件的可点击演示
- 8 个实战案例页面（登录、商品列表、聊天……）
- 138 道测试题的答题系统
- 24 项硬件能力（扫码、定位、振动……）的实时体验

> 全部由 AI 生成，你要做的只是**用大白话把需求讲清楚**，然后学会**在微信开发者工具里把它跑起来、调好、发出去**。

## 核心理念：什么是 vibe coding？

传统开发是「人写代码，电脑执行」。
**vibe coding** 是「人说需求，AI 写代码，人负责验收和调试」。

这套教程每一章都遵循同一个节奏：

<div class="note">
<b>你说的话</b> → <b>AI 做的事</b> → <b>你在开发者工具里看到的结果</b> → <b>出问题怎么让 AI 修</b>
</div>

你不需要懂 `.js`、`.wxml` 是什么，只需要学会**看效果、发指令、报 bug**。
