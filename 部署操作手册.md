## 第四步：上传发布

### 4.1 上传代码

1. 点击工具栏上的 **"上传"** 按钮
2. 填写版本号（如 `1.0.0`）和项目备注
3. 点击确定，等待上传完成

### 4.2 提交审核（正式发布需要）

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)（用测试号绑定的微信扫码登录）
2. 进入 **"管理" → "版本管理"**
3. 在"开发版本"中找到刚上传的版本
4. 点击 **"提交审核"**
5. 审核通过后，点击 **"发布"**

> 注意：测试号有一些限制，无法正式发布到线上供所有用户使用。测试号主要用于开发和调试阶段。如果要正式上线，需要注册正式的小程序账号。

## 第五步：常见问题排查

### Q: 编译报错怎么办？

1. 查看控制台报错信息
2. 常见原因：
   - 文件路径大小写不一致
   - JSON 格式错误（多了逗号、少了引号）
   - 组件未在 json 中声明却在使用
3. 把报错信息发给我，我帮你修

### Q: 模拟器白屏？

1. 检查 Console 是否有 JS 报错
2. 尝试点击工具栏 **"编译"** 重新编译
3. 清除缓存：菜单栏 **"清缓存" → "全部清除"**，然后重新编译

### Q: 真机预览打不开？

1. 确保手机微信是最新版本
2. 确保手机和电脑网络畅通
3. 测试号需要微信账号被添加为体验者（在公众平台的"成员管理"中添加）

### Q: 某些组件显示异常？

1. 检查对应组件的 wxml/wxss 是否正确
2. 检查 data/controls.js 中对应组件的 id 是否和详情页传参一致
3. 把异常截图发给我
